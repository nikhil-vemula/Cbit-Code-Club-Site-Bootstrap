What is HTML?

Hyper Text Markup Language(HTML) is used to create Web pages that can be displayed in any browser. Lets see how it looks like, open your browser, open any web page or website right click> view page source you can see hundred of lines of Html code. Don't worry most of it is generated using some framework or software. In order to use such frame works we need to have idea about html.

Let's start with a basic example and then understand it


<!DOCTYPE html>
<html>
<title>My first webpage</title>
<head>
</head>
<body>
Hello world
</body>
</html>
</code>

The tag <!DOCTYPE html> defines this document to be HTML5. A Html document has mainly head and body parts. Head specifies the meta information of the page like title,keywords,refresh rate etc. Body is where the content of the html page is placed.

Tag:The elements like <html>,<head>,<body> etc are called Tags, they are used to structure the html document.
  1. Tag with start and end tag 
    <tag> -- start tag
</tag> -- end tag
example : <body> </body>
  2. Tag without end tag 
    <tag>
example: <br>
Attributes:A html tag may have may have various attributes to specify the data or style of that element 
Example:  <img src="image.jpg" />
Here src is the attribute and image.jpg is the attribute value
